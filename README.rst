SCALES-DRP: the SCALES data reduction pipeline
=======================================

Install
.......
git clone git@github.com:Keck-DataReductionPipelines/KCWI_DRP.git

cd SCALES-DRP

pip install -e .

Quick start
...........

Next, go to the data directory and run the startup script:

cd mydata

reduce_kcwi -lr -l target.txt
